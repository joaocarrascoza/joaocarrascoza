Projects in College

These projects involve the practical application of concepts learned in college. I used programming languages such as C, Python, and Java to develop them. They were all done in groups, each with a brief explanation of its purpose and followed by its respective source code.

System of a Hotel

As my first project in college, I developed a hotel management system in C using initial concepts. This system includes various functionalities, such as check-in, check-out, among others, as well as allowing the user to make reservations, view room occupancy, cancel reservations, view hotel occupancy rate, and register and view guest information. Click here to see the source code.

Image Processor

This project implements an image processor with a graphical user interface (GUI) in Python. The application allows the user to perform the following basic operations on the image: create a histogram of the image, convert the image to black and white, apply two selected types of filters and save the altered images.
